# 人物管理

## 運用ルール

このファイルは、`校正後プロット.md` を本文化する際の人物設定、口調、役割、秘密、変化を管理するための台帳である。

### 基本方針

1. すべての人物は「見てほしい」「見ていたつもり」「見誤った」のいずれかを背負う。
2. 悪人を記号化しない。
3. 被害者を純化しない。
4. 誰か一人に物語の正解を背負わせない。
5. 人物の変化は、思想ではなく行動で見せる。

## 神崎 澪

### 基本情報

| 項目 | 内容 |
|---|---|
| 年齢 | 23歳 |
| 立場 | 地下アイドル、主人公 |
| 物語上の役割 | 見られる側から、真相を見る側へ変わる人物 |
| 最終到達点 | 見られるためだけではなく、会い続けるために歌う |

### 人格

- 皮肉屋。
- 理屈っぽい。
- 綺麗事が嫌い。
- 人間観察が好き。
- 自分の承認欲求を軽蔑している。
- しかし誰よりも見つけてほしい。
- かわいそうなだけの主人公ではない。
- 時に他人を傷つける。

### 欠落

自分には価値がないと思っている。

そのため、数字、チェキ列、視線、反応で自分を測ってしまう。

### 欲望

- 認められたい。
- 見つけてほしい。
- でも、見られすぎるのは怖い。
- 代用品として見られることには強く反発する。

### 物語上の変化

| 段階 | 状態 |
|---|---|
| 第一部 | 見てほしいが、見られることを恐れる |
| 第二部 | 蒼や周囲を見る側に回る |
| 第三部 | 信じたい人を疑う勇気を持つ |
| 第四部 | 自分の加害性と承認欲求を認める |
| 第五部 | 黒崎の絶望を否定しきれない |
| 第六部 | 理解されなくてもよいと言える |
| 第七部 | 理解できない他者へ歌う |

### 口調

- 基本は短め。
- 皮肉が混じる。
- 傷つくと理屈っぽくなる。
- 本当に苦しいときは言葉が減る。

### 例

> 「好きでやってるように見える？」

> 「嫌いだよ。嫌いだけど、やめられるほど嫌いじゃない」

> 「誰も私を理解しなくていい。私も誰も理解できないから」

### 禁止事項

- かわいそうな被害者としてだけ描く。
- 正義の主人公にする。
- 蒼に救われる構図にする。
- 承認欲求を完全に克服させる。

### 欲望

- 誰かを救いたい。

- 澪を守りたいが、それが十年前のやり直しなのか自分でも分からない。

### 物語上の変化

- 完璧な保護者にする。
- 恋愛相手として澪を救わせる。

### 人格

- 承認欲求が強い。

- 弱い。

### 欲望

- 有名になりたい。
- 愛されたい。
- 見つけてほしい。
- でも、自分抜きで自分の物語を語られたくない。

### 物語上の機能

| 見え方 | 実態 |
|---|---|

> 「私抜きで、私の話をしないでほしい」
